# [ChemWaR]()

![Python](https://img.shields.io/static/v1?label=python&message=3.11.2&color=blue) 
![Flask](https://img.shields.io/static/v1?label=flask&message=2.2.0&color=092E20)

![GitHub repo size](https://img.shields.io/github/repo-size/Natte2110/chemwar?color=orange) ![GitHub pull requests](https://img.shields.io/github/issues-pr/Natte2110/chemwar)

This project has been created in order to provide a central hub for responding to HADR events across the UK.

The site enables users to create, manage and update cordons around dangerous events.
